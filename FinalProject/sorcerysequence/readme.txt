# CIT190 Final Project: Wizards Arcade

-------------------------------------------
## Game Information
-------------------------------------------
**Sorcery Sequence**: Test your magical memory and reflexes as you match colors in sequence in this twist on the classic game 'Simon Says'.
------ Game Purpose: 
            Similar to the classic game 'Simon Says', this version will challenge your magical memory and puts your reflexes to the test as you strive to match colors in sequence.
            Colored buttons will light up in a specific sequence. Pay close attention to the order in which they light up, then click the buttons in the same order to match the sequence.
------ Game Controls:
            Use your mouse to click on the colored buttons displayed on the screen. Click each button in the same order as they light up to match the sequence.
------ Future Release:
            Enhance graphics and styling.
